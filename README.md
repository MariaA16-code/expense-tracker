# Expense Tracker 💸

A simple expense tracking web app built with Flask and MySQL.

## Features
- Add expenses with title, amount, category and date
- Delete expenses
- Filter by category
- Live total summary

## Tech Stack
- Backend: Flask, SQLAlchemy
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## Setup & Run

1. Clone the repo
git clone https://github.com/MariaA16-code/expense-tracker

2. Activate virtual environment and install dependencies
pip install -r requirements.txt

3. Create MySQL database
CREATE DATABASE expense_tracker;

4. Update database password in app.py

5. Run the app
python app.py

6. Open browser
http://127.0.0.1:5000
